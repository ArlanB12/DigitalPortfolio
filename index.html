Hi5 Portal
project_01 · Capstone · STI College Naga
About This Project

Hi5 Portal is a full-stack capstone project for DSPMNHS. An integrated enrollment and academic records management system with an AI-powered at-risk student prediction module using Python and scikit-learn.

Tech Stack
React 18TypeScriptVite 6Tailwind CSS 4Radix UIRechartsLaravelPostgreSQLPythonscikit-learn
Key Features
  • AI-powered at-risk student prediction using machine learning
  • Auto-sectioning with gender and classification awareness
  • Grade upload and SF9-compliant academic records management
  • Role-based access: Admin, Registrar, Teacher, Student
  • Account lockout after failed login attempts
  • Dark mode, unified registrar theme, toast notifications

About This Project

Alok (Offer) is a short film on professional issues in Information Systems and Technology, directed by Jheamuel Panuelos and produced by Comflix Studios. The film follows an employee who is introduced to an online gambling site. What begins as casual enjoyment gradually turns into addiction and distraction, ultimately compromising mental health, work performance, and personal priorities. Alok highlights the dangers of unchecked digital temptations and the professional issues that arise when technology intersects with human vulnerability.
